Adds gui and workflow to ofxColorQuantizer. 

Easy to include ofxColorQuantizer in your app project.

## Overview
* drag images into window
* set the desired number of colors
* sort colors by H/S/B

## Dependencies

ofxColorQuantizer

ofxColorQuantizerHelper

ofxGui

ofxOpenCv
